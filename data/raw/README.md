# Raw Data

Do not commit raw DICOM files to GitHub. Download TCIA collections locally and place them in the following folders when running full imaging experiments:

```text
data/raw/duke/
data/raw/ispy1/
data/raw/ispy2/
data/raw/acrin6698/
```

The GitHub repository should contain code, derived feature tables, documentation, and results, not full DICOM archives.
